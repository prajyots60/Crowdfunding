# Channel 1 UI Demo

This folder contains a minimal UI + backend to demo the Channel 1 flow (register, validate, create project, approve, fund, query).

## 1) Start backend

```bash
cd /home/supra/Desktop/Crowdfunding/ui-channel1/backend
npm install
npm start
```

Backend runs at http://localhost:4000

## 2) Open UI

Open this file in a browser:

- /home/supra/Desktop/Crowdfunding/ui-channel1/frontend/index.html

## 3) Demo flow

1. Register Startup
2. Validate Startup
3. Register Investor
4. Validate Investor
5. Create Project
6. Approve Project
7. Fund Project
8. Get Project

Note: Microfab must be running and wallets/MSP must be generated before using the UI.
